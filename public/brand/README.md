# TestoCMS Brand Assets

This directory contains the clean SVG brand pack derived from the selected TestoCMS toast mark concept.

Files:

- `testocms-mark.svg` — canonical icon-only mark
- `testocms-app-icon.svg` — square app icon version with extra safe area
- `testocms-logo-header.svg` — compact horizontal logo for website headers
- `testocms-logo-readme.svg` — wider horizontal logo for README and docs covers

The browser favicon asset lives at:

- `/favicon.svg`
