# Support

## Community Use

Use GitHub issues for reproducible bugs and feature requests.

## Private / Commercial Contact

For private inquiries, commercial discussions, or maintainer contact, email [me@ilindberg.ru](mailto:me@ilindberg.ru).

## Before Opening an Issue

- confirm the behavior on the latest main branch or latest tagged release
- include environment details
- include logs, stack traces, screenshots, or request samples when relevant

## Security

Use [SECURITY.md](SECURITY.md) for vulnerability disclosure. Do not post security issues publicly.
